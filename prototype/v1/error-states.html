💡 Common Issues & Solutions

Server Connection Failed
▼

This usually happens when the server is not running or the URL is incorrect.

  1. Verify server is running: ps aux | grep tanaka
  2. Check server logs: tail -f ~/.local/share/tanaka/tanaka.log
  3. Test connection: curl https://localhost:8443/health
  4. If using self-signed cert, accept it in browser first
Authentication Failed
▼

Your auth token doesn't match the server configuration.

  1. Check token in server config: ~/.config/tanaka/tanaka.toml
  2. Update token in extension settings
  3. Ensure no extra spaces in token
  4. Restart server after config changes
Sync Stuck or Delayed
▼

Large numbers of operations can cause sync delays.

  1. Check operation queue size in debug info
  2. Force sync: Click "Sync Now" in popup
  3. Clear cache if queue is corrupted
  4. Increase sync frequency in settings
Missing Permissions
▼

Firefox may have revoked extension permissions.

  1. Go to about:addons
  2. Click on Tanaka extension
  3. Select "Permissions" tab
  4. Enable all required permissions
